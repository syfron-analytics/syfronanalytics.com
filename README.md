# Syfron Analytics — Landing Page

Static landing page built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:4321`

## Build

```bash
npm run build
```

Output in `dist/`. Serve with any static host.

## Preview

```bash
npm run preview
```

## Features

- 🌍 Spanish/English toggle (client-side i18n)
- 🌙 Dark mode design
- 📱 Responsive
- ⚡ Static output, zero JS frameworks
